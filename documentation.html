Discord Bot Documentation

A community bot is planned for users. Features include game info, report submission, and limited self-service tools.

Commands

/game

Sends an embed with game info, platform availability, and macro keys.

/prefix <symbol>

Sets a custom prefix per server.

,report <userId> <reason>

Submits a report to staff review.

Webhooks

Roblox can send event logs to a dedicated channel using a Discord webhook URL. Events include player joins, auto-kicks for young accounts, and staff actions.

API Bridge

An optional API bridge allows actions initiated in Discord to reflect in-game. Authentication is required and actions are rate-limited.
